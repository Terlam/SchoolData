Illinois School Data API is a rest-based web service. That allows the display of data from the Illinois Report Card Data Library .

GET/schools

GET/schools/<rctds code>

- Retrieves all schools

- Retreives a school by it's RTCDS number

In order to gain access to the API users must register and request an API Token. Tokens are valid for up to one hour after request and then each request must be renewed. To retrieve your token register for the site.
